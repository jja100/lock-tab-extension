# Lock Tab Extension

A simple Chrome extension that lets you lock and unlock browser tabs to prevent accidental closure or navigation. Features a dynamic toolbar icon and popup UI for easy tab management. Perfect for keeping important tabs safe while you browse.

## Features
- Lock/unlock any tab with a single click or keyboard shortcut
- Prevents accidental tab closure or navigation
- Dynamic extension icon reflects lock state
- Simple popup UI for quick access

## Installation
1. Clone or download this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select this folder.

## Usage
- Click the extension icon to open the popup.
- Use the **Lock Tab** button to lock or unlock the current tab.
- The icon will change to indicate the lock state.
- You can also use the keyboard shortcut (default: `Ctrl+Shift+Z`).

## Building & Packaging
To pack the extension for distribution:
1. Ensure all files are present and up to date.
2. Zip the folder or use Chrome's **Pack extension** feature in `chrome://extensions/`.
